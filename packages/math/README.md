Simple linear algebra Math library to support [chemistry js project](http://vreshch.com/chemistry-js.html)

## Include following clases:
  * Vec3
  * Matrix3x3
  * Matrix3x4
  * Transform3d
  * Quaternion

## Commands:
  * Build project: `npm run build`
